# Pelee SSD Replication

This repository contains the PyTorch implementation of the Pelee object detection architecture.

## Setup Instructions

1. In a folder pretrain, download the pretraining pelee from the github repository https://github.com/yxlijun/Pelee.Pytorch.
2. Download the VOCdevkit from kaggle: https://www.kaggle.com/datasets/wangyuhang3303/vocdevkit?resource=download
3. python train.py
